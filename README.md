# task
UI Automation (Playwright / Cypress)
What to build
Automate a user flow on the Careerflow.ai web application or any publicly accessible web app of your choice (e.g., login, sign-up, form submission, job tracker interaction).
Use Playwright or Cypress. Structure your code using the Page Object Model (POM).
Requirements
Minimum one complete end-to-end flow with at least 3 steps
Meaningful assertions at each step — not just navigation checks
No hardcoded waits — use proper async/synchronisation mechanisms
Code must be readable, modular, and reusable
Include a brief comment in the test file explaining your choice of flow and why it matters
Bonus
Integrate this into a GitHub Actions workflow that runs on push
